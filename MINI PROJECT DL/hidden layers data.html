    Handwritten Digit Recognition by using CNN

    We can recognize the hand written digits from the famous mnist dataset by using CNN.

    1. Import Dataset
    2. Data Preproessing
    3. Build CNN Model

    1. Import Dataset

    Import the mnist dataset from the keras library.

    The input image have 28 X 28 pixels.

    Single handwritten image in the dataset.

    2. Data Preproessing

    Reshape

    Reshape the input image into 1 dimension for the input layer of the neural networks.

    Set the input shape of the image is (28,28,1) and the input data as the type of float32.

    Normalize the data

    The pixels values is change from the range of 0 – 255 to 0 – 1 for the better accuracy of the neural networks.

    Now the all pixel value is changed within the range of 0 – 1.

    One Hot Encoder

    One Hot Encoder for the labels of the mnist dataset.

    Now the label is have 10 values, it represented the digit from 0 – 9 .The input image digit position is have the value 1 and all other values is 0 in the label.

    Set the number of pixels and number of classes.

    3. Build CNN Model

    Import all packages

    The required packages are imported from the keras library. Then create the object for the keras model.

    CNN Layers

    Build the CNN layers.One way to improve the performance a Neural Network is to add more layers.

    we use relu activation for the input layer and softmax activation for the output layers.

    Compile the model

    The SGD optimizer is used with the learning rate is 0.01 for compile the neural network. The model summary shows the details of the neural network.

    The input image pixels is 28 X 28 but the first convolutional layer input pixels is 26 X 26 because of the filters. The second layer have the 24 X 24 pixels.

    The filter is reduce the pixels because we don’t use the padding.The max pooling is reduce the pixels from the convolutional layer.

    Train the model

    we can train the model with 10 epochs, i.e 10 iteration of the input data.

    Evaluate the model

    The accuracy of the Convolutional Neural Network model is 98%.

    Save and Load the model

    Predictions

    The predicted handwritten digit is Seven(7).
